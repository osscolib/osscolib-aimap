
 Building osscolib-indexmap
 --------------------------
 
 To build osscolib-indexmap you will need Maven 2. You can get it at:
 
     http://maven.apache.org

 Build and install the project executing, from the osscolib-indexmap
 project root folder:
 
     mvn clean:clean install
     
 And you will get a fresh target/osscolib-indexmap-{version}.jar file.
 You will also get it installed in your local repository at:
 
     $M2_REPO/org/osscolib/osscolib-indexmap/{version}/osscolib-indexmap-{version}.jar



 Generating Javadoc for osscolib-indexmap
 ----------------------------------------
 
 If you wish to generate the javadoc for osscolib-indexmap, execute this from the
 osscolib-indexmap root folder:
 
     mvn javadoc:javadoc
     
 This will generate the javadoc documentation in HTML format in:
 
     target/site/apidocs
